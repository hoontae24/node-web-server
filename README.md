# node-web-server

- 네트워크 연결 확인 용 간단한 노드 서버
- 적용 가능한 환경변수
  - `PORT`: Http 서버 포트
  - `DB_HOST`: 데이터베이스 호스트
  - `DB_USER`: 데이터베이스 유저명
  - `DB_PASSWORD`: 데이터베이스 비밀번호
  - `DB_PORT`: 데이터베이스 포트
  - `DB_NAME`: 데이터베이스 명
- 실행:
  - `npm install` -> `npm start`
